# Agent Instructions: CS202 Seminar - Facade Design Pattern

## Context & Objectives
This repository is for a CS202 (OOP) university seminar project on the **Facade** design pattern. The goal is to produce:
1. A Markdown report.
2. A presentation outline.
3. Source code demonstrating both a naive solution and the Facade pattern solution.
4. A multiple-choice quiz (5–8 questions).
5. A list of references.

## Deliverables & Organization
- **Report**: Create as `REPORT.md` in the root. Ensure it covers all 11 specific requirements from `README.md` in order.
- **Class Diagrams**: Use **Mermaid.js** syntax within `REPORT.md` to draw the class diagrams (the general Facade pattern and the problem-specific diagram).
- **Source Code**: Put all source code in a `src/` folder. Implement:
  - A naive solution (without the pattern).
  - The pattern-based solution (with the Facade).
- **Language Choice**: Verify the preferred OOP language (typically C++, Java, or C# for CS202) with the user before implementing.
